# US-3 Logout

**Owner:** Omar Eshatti  
**Technology Stack:** JavaScript + Node.js

## User Story
As a logged-in user,  
I want to log out,  
so that my account is not accessible on this device.

## Acceptance Criteria
- Logout button exists
- Authentication token/session is cleared
- User is redirected to login or home page
- Protected pages are blocked after logout
