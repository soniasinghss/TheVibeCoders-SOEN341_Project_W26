# MealMajor  
A Meal Prep and Healthy Living Planner

## Project Description  
MealMajor is a web application made for students to help them plan meals, keep track of groceries, and find simple and healthy recipes. The purpose of this project is to practice software development skills while building a functional prototype using Agile methods.

This project is part of a team assignment for Winter 2026 and is developed in four sprints following the Scrum framework.

## Baseline Features  

### 1. User Account Management  
- User registration and login.  
- Profile management, including diet preferences and allergies.

### 2. Recipe Management  
- Create, edit, and delete recipes.  
- Each recipe includes:
  - Ingredients  
  - Preparation time  
  - Preparation steps  
  - Cost  
- Search for recipes.  
- Filter recipes based on:
  - Time  
  - Difficulty  
  - Cost  
  - Dietary tags (e.g. vegetarian, vegan, gluten-free).

### 3. Weekly Meal Planner  
- Create a weekly meal plan.  
- View meals in a weekly grid.  
- Assign recipes by:
  - Day of the week  
  - Meal type (breakfast, lunch, dinner, snack).  
- Edit or remove meals from the planner.  
- Prevent duplicate meals within the same week.
